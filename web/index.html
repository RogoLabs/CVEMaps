About CVE Maps

Data Source

Parsed from the official CVE List V5 repository. Analyzing CVEs published in the last 365 days with CNA assignments and CWE mappings from both primary sources and authorized data publishers (ADPs).

Auto-Updates

Automatically rebuilds every 3 hours via GitHub Actions, ensuring you always see the latest vulnerability landscape and emerging patterns from recent CVEs.

CNAs

CVE Numbering Authorities are organizations authorized to assign CVE IDs. Examples include MITRE, Oracle, Microsoft, and hundreds of others.

CWEs

Common Weakness Enumerations categorize software vulnerabilities (e.g., CWE-79: Cross-Site Scripting, CWE-89: SQL Injection).

Visualizations

14 interactive network graphs covering CNA-CWE relationships, co-occurrence patterns, vendor profiles, CVSS severity analysis, temporal trends, and more. All visualizations support zoom, pan, and click-to-highlight interactions.

Network Analysis

Built using NetworkX and D3.js force simulations. Graph layouts include force-directed, bipartite, hierarchical tree, circular, star, and ego network arrangements optimized for pattern discovery.

Rolling Window

365-day rolling window keeps visualizations focused on current threats. As new CVEs are published, older ones roll off, maintaining a consistent view of the active vulnerability landscape.

Open Source

Fully open source under MIT License. Fork, modify, and contribute on GitHub. All build scripts and data processing pipelines are available for transparency and reproducibility.
